# 🚀 Инструкция запуска монитора заказов

## Шаг 1 — Создай Telegram бота (5 минут)

1. Открой Telegram → найди @BotFather
2. Напиши /newbot
3. Придумай имя: например "Мои заказы"
4. Придумай username: например "moi_zakazy_bot"
5. Скопируй токен — вида: 7123456789:AAF_xyz...

Узнай свой chat_id:
1. Найди @userinfobot в Telegram
2. Напиши /start
3. Скопируй число из поля "Id"

Вставь оба значения в monitor.py:
   TELEGRAM_TOKEN = "7123456789:AAF_xyz..."
   TELEGRAM_CHAT_ID = "123456789"


## Шаг 2 — Бесплатный сервер (Railway.app)

Это самый простой способ — бесплатно, без карты, работает 24/7.

1. Зайди на railway.app
2. Войди через GitHub
3. New Project → Deploy from GitHub repo
4. Загрузи папку freelance-monitor на GitHub
5. Railway сам установит зависимости и запустит

Альтернатива — Render.com:
1. render.com → New → Background Worker
2. Подключи GitHub репозиторий
3. Build Command:  pip install -r requirements.txt
4. Start Command:  python monitor.py
5. Бесплатный план — работает 24/7


## Шаг 3 — Запуск локально (для теста)

# Установить зависимости
pip install -r requirements.txt

# Запустить
python monitor.py


## Шаг 4 — Настройка под себя

Открой monitor.py и измени KEYWORDS под свои услуги:

KEYWORDS = [
    "сайт", "лендинг", "верстка",      # базовое
    "wordpress", "bitrix", "modx",       # CMS
    "react", "vue", "frontend",          # фронтенд
    "php", "python", "backend",          # бэкенд
    "интернет-магазин", "магазин",       # коммерция
]

Чем точнее ключи — тем меньше мусора в уведомлениях.

## Частые вопросы

Q: Не приходят уведомления?
A: Проверь токен и chat_id. Напиши боту /start перед запуском скрипта.

Q: Много мусорных заказов?
A: Сузь KEYWORDS, добавь слова точнее.

Q: Авито не парсится?
A: У Авито есть защита — это нормально, остальные три площадки работают стабильно.

Q: Как добавить новую площадку?
A: Скопируй любую функцию parse_xxx(), измени URL и селекторы CSS.
